# Java OOP & Data Structures

A learning repository for Object-Oriented Programming concepts and Data Structures implementation in Java.

## Project Structure

```
java-oop-data-structures/
├── ListaSimple/          # Singly Linked List implementation
│   └── src/
│       ├── Lista.java    # Linked list class
│       ├── Nodo.java     # Node class
│       └── Main.java     # Demo program for student grades
```

## Current Implementations

### 1. ListaSimple (Singly Linked List)

A simple linked list implementation that demonstrates:
- Node-based data structure
- Dynamic insertion
- Position-based search
- List traversal
- Average calculation

**Features:**
- Insert values at the end of the list
- Search nodes by position
- Display all list contents
- Calculate average of stored values

**Example Use Case:**
The current demo allows users to input student grades and calculates the final average.

## How to Run

### ListaSimple
```bash
cd ListaSimple/src
javac *.java
java Main
```

Or open the project in your favorite Java IDE (IntelliJ IDEA, Eclipse, VS Code).
