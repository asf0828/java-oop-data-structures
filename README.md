# Java OOP & Data Structures

A learning repository for Object-Oriented Programming concepts and Data Structures implementation in Java.

## Projects

- [ListaSimple](./ListaSimple/) - Singly Linked List implementation
- [OperacionesLIstas](./OperacionesLIstas/) - List operations: count characters, compare lists, and remove duplicates

## Requirements

- Java JDK 8 or higher
- Any Java IDE or text editor

## Learning Goals

This repository is designed to practice:
- Object-Oriented Programming principles
- Data structure implementation from scratch
- Algorithm design and analysis
- Java programming fundamentals
